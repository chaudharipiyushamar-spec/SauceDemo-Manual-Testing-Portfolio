# Sauce Demo Manual Testing Project

## Project Overview
This project demonstrates a comprehensive manual testing process for the **Sauce Demo (Swag Labs)** e-commerce application. It covers the full Testing Life Cycle (STLC), from requirement analysis to test closure.

## Documentation Included
- **Requirement Analysis:** Breakdown of functional and non-functional requirements.
- **Test Plan:** Strategy, scope, environment, and exit criteria.
- **Test Scenarios & Cases:** 28+ test cases covering Login, Inventory, Cart, and Checkout flows.
- **Test Closure Report:** Summary of execution results and environment details.

## Tools Used
- Documentation: PDF, MS Excel
- Testing Site: [SauceDemo](https://www.saucedemo.com/)
- Browser: Chrome, Firefox, Edge

## Key Skills Demonstrated
- Functional & Negative Testing
- UI/UX Validation
- Test Case Design
- Bug Reporting & STLC Documentation
